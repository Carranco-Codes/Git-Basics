# üìñ Terms
| Term          | Definition                                                                                                                              |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| Clone         | When you clone a repository, you create a local copy of a repository on your computer to sync with the remote repository on GitHub.     |
| Fork          | A fork is a copy of a repository that allows you to make your changes without impacting the original project.                           |
| Pull Request  | Pull requests let you tell others about changes you've pushed to a branch in a repository on GitHub.                                    |
